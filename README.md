# Page Visit Counter

Эта система позволяет собирать статистику посещений с любого сайта (IP, Город, Устройство) и отображать её на удобном дашборде. Система построена на базе Laravel и SQLite.

## Запуск проекта

Так как проект использует встроенную базу данных SQLite, дополнительная настройка СУБД не требуется. 

1. Убедитесь, что у вас установлен PHP и Composer.
2. В корневой директории проекта установите зависимости (если они ещё не установлены):
   ```bash
   composer install
   ```
3. Запустите встроенный сервер Laravel:
   ```bash
   php artisan serve --host=127.0.0.1 --port=8000
   ```

## Как использовать счетчик на своем сайте

Чтобы начать отслеживать посещения на любом сайте, добавьте следующий тег `<script>` перед закрывающим тегом `</body>` на страницах вашего сайта:

```html
<script src="http://127.0.0.1:8000/tracker.js"></script>
```

> **Примечание:** Если вы развернете систему на реальном сервере, замените `http://127.0.0.1:8000` на ваше реальное доменное имя. Также потребуется обновить адрес внутри самого файла `public/tracker.js`.

## Просмотр статистики

Все собранные данные можно просмотреть на специальном дашборде. 

1. Откройте в браузере: **http://127.0.0.1:8000/stats**
2. При запросе авторизации введите следующие данные:
   - **Логин:** `admin`
   - **Пароль:** `admin`

На странице вы увидите линейный график посещений за последние 24 часа и круговую диаграмму распределения визитов по городам.

## Тестирование

В проекте предусмотрена тестовая страница для проверки работы счетчика. Просто откройте корневой адрес проекта в браузере:
**http://127.0.0.1:8000/**

При открытии этой страницы ваш визит будет немедленно засчитан и появится на дашборде.
